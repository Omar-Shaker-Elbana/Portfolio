Dijkstra's Algorithm
Completed

A clean, production-ready implementation of Dijkstra's shortest path algorithm built from scratch in Python — no library calls, just a custom Graph class, a min-heap priority queue, and path reconstruction logic. Validated against a hand-crafted 17-node graph with cycles, zero-weight edges, chords, and a fully disconnected island to stress-test correctness at the edges.

  • Custom Graph class — adjacency list, add/remove vertex & edge, vertex renaming
  • O((V + E) log V) via binary heap — handles 0-weight edges and unreachable nodes
  • Path reconstruction with full route tracing back to source
  • 17-node stress-test graph with cycles, traps, chords, and disconnected components
